# 파이썬의 특징

- 파이썬의 특징
  - 인간다운 언어다
  - 쉽게 배울 수 있다
  - 무료다
  - 간결하다
  - 프로그래밍을 즐기게 해준다
  - 개발 속도가 빠르다
  - 제가 쓴거 아닙니다 (출처 : https://wikidocs.net/17684)

- 파이썬은 한줄씩 실행하는 인터프리터 방식의 언어다
  - C 같은 컴파일러방식(기계어로 미리 변환)이 아닌 런타임에 소스를 기계어로 변환
  - 당연히 런타임에 변환이 필요하므로 매우 느리나 (심각하게 느림) 실행단계에서 소스의 수정이 가능


# 변수와 상수

- 상수란? 
  - 변하지 않는 수
  - $\pi$, 155, '고양이' 등 우리가 사용하는 대부분의 값
- 변수란?
  - 변할수 있는 수로 값을 저장하기 위해 사용
  - 변수 선언방법
    - 값 할당시 자동으로 생성 됨
    
    ```
    a1 = 301    # a1 변수를 선언하고 301의 값을 저장
    a2 = 401    # a2 변수를 선언하고 401의 값을 저장

    a1 = a1 + 1     # a1에 있는 값과 1을 더하여 a1에 저장
    ```
    
    - 일반적으로 프로그래밍 언어에서 = 기호는 같다가 아닌 대입의 의미
      - 우측에 있는 값을 좌측에 대입한다는 의미로 보면 됨

  - 변수 명명 규칙 
    - 규칙은 꽤 관대하지만 널리 사용되는 관례? 를 어느정도 따르는게 좋음
    - 대소문자가 구분되므로 혼동이 없도록 규칙을 정해서 사용 
      - 스네이크표기법 : 소문자만 사용하고 띄어쓰기는 _로 대체
      - 일반적인 변수를 _로 시작하는것 절대 금지
    - 이름만으로 변수의 기능을 최대한 알 수 있도록 명명
    - 내장된 함수 나 변수, 패키지 등의 이름 절대 사용 금지
  - type() 함수를 사용하여 변수의 타입을 알 수 있음
  

## 숫자형 변수

- 정수형 (integer)
  - -11, 0, 12 등 정수형 데이터를 저장
  - 다른언어와 크게 다르게 ***자리수의 제한이 없음***
  - 선언 방법
    ```
    a1 = 3
    a2 = -55
    ```

- 실수형 (float)
  - 소수점이 포함된 숫자 데이터를 저장
  - 자리수의 제한이 있음 (정수부와 소소부를 합쳐 약 17자리)
  - 선언 방법
    ```
    a1 = 3.333
    a2 = -55.
    a3 = .2
    ```    
- 숫자형의 연산
  - 기본적으로 사칙연산 사용가능
    ```
    a = 3 + 3
    b = a - 3
    a = b * 5
    b = b / 2
    ```    
  - 거듭제곱, 몫과 나머지 계산 가능 
    ```
    a = 2 ** 3
    b = a // 3
    a = a % 5
    ```    
  - 같은 타입끼리 (int와 int 혹은 float와 float) 연산시 같은 타입이 나옴
  - int와 float 을 연산시 float으로 출력됨(계산전 int쪽이 float로 변환 발생)

## 문자열 변수
- 문자열 (string)
  - 우리가 사용하는 모든 문자를 사용가능
  - 다른언어에서는 이 문자열처리가 굉장히 어려운데 반해 파이썬은 간단하게 사용할 수 있는 구조를 가짐
  - 기본적으로 소스코드 와 문자열 변수에 대해 유니코드(UTF-8)를 지원해 대부분의 언어가 사용 가능

  - 선언 방법
    - ' 혹은 " , 멀티라인일 경우 ''' 혹은 """ 중 아무거나 사용 가능
    ```
    a = 'apple'
    b = "나는 apple을 좋아한다"
    a = 'الأرنب = খরগোশ = खरगोश = กระต่าย'
    खरगोश = '''
    첫번째줄은 나는 길을 걷는다
    두번째줄
    '''
    ```    
    - 물론 가능할뿐이지 변수명은 영어로...(에디터에서 지원이 안될 수 있음)
- 문자열 연산
  - 문자열 연결(+), 문자열반복(*)
    ```
    a = "Add" + "Add"       # AddAdd
    b = "Add" * 3           # AddAddAdd
    ```
- F스트링
  - 변수에 있는 값을 문자열에 넣고 싶을때 사용하는 formatting 방법중 하나
  - 파이썬 3.6부터 생긴방식으로 매우 편하다 (이전 버전에선 지원 안함)
    ```
    a1 = '요구르트'
    b1 = f'{ a1 }는 맛있다'       # 요구르트는 맛있다
    a1 = '야쿠르트'
    b1 = f'{ a1 }는 맛있다'       # 야쿠르트는 맛있다
    a2 = 3
    b1 = f'강도 {a2}명이 {a1}를 먹는다'     # 강도 3명이 야쿠르트를 먹는다
    ```
- 슬라이싱
  - 리스트처럼 위치값이나 슬라이싱을 사용해 일부 추출이가능
    - 대신 문자열은 immutable한 객체라 수정은 불가능
  - 자세한 내용은 뒤에 리스트에서 언급

## 논리형 변수 (bool)
- 참과 거짓을 표현하는 데이터를 저장
  - 사실상 컴퓨터에서 표현가능한 유일한 변수
  - True(사실), False(거짓) 두가지의 값을가짐 (첫글자 대문자 주의)
  - 조건문 등에 주로 사용됨 (밑에 나오는 조건문은 연산후 논리형을 반환함)
  - 참고로 None 이라는 값과는 다른값임(이것은 없다를 의미)
- 다른 형태와의 변환
  - 숫자형을 논리형으로 변환시 0이 아니면 True, 0이면 False
  - 문자형을 논리형으로 변환시 ''(비어있음)이 아니면 True, '' 면 False
  - 논리형을 숫자형으로 변환시 True 는 1, False 는 0 
  - 논리형을 문자형으로 변환시 True 는 'True', False 는 'False' 
  - None을 논리형으로 변환시 False ( None이 아닌객체는 대부분 True )



## 리스트
- 리스트란?
  - 변수 하나에 한개가 아닌 그 이상의 데이터를 담을 수 있도록 만든 구조
    - 형태에 상관없이 모든 변수를 원소로 가질 수 있음
    - 물리적으로 선형의 구조를 가지나 **값이 아닌 Reference**를 저장
  - 인덱싱
    - 여러개의 원소에 위치번호를 사용하여 접근 가능
    - 1번쨰 자리는 0부터 시작하므로 1을 더해서 계산( 5는 6번쨰자리 )
    - [-1]은 맨뒤를 의미
  - 슬라이싱
    - 연속적인 값을 한번에 추출가능
    - [3:5] : 3번위치(4번째) 부터 시작하여 5번위치(6번째) **앞**까지 선택
    - [:3], [3:] 등 미지정시 해당 방향 전체를 의미 ([:]는 전체)
    - [3:9:2] : 3번째 요소를 사용하여 점프 가능 (미입력시 1)
    - 슬라이싱 결과는 무조건 리스트로 출력(결과가 1개더라도)
    ```
    li = []                     # 빈 리스트 선언
    li = [1,2,3,4,5,6,7,8,9]    # 3개의 원소를 가진 리스트 선언
    li[0]                       # 1번째 : 1
    li[0:1]                     # 1번째 : [1]
    li[2:4]            # 3번째 부터 5번째 이후 제거 : [3,4]
    li[:-1]            # 처음부터 마지막칸 제거 : [1,2,3,4,5,6,7,8]
    li[0::2]           # 1번째 부터 마지막까지 2칸씩 이동하며 선택 : [1,3,5,7]
    ```
- 리스트 연산
  - 연결(+)과 반복(*)은 문자열과 같음
    ```
    a = [1,2,3]         # [1,2,3]
    a + a               # [1,2,3,1,2,3]
    a * 3               # [1,2,3,1,2,3,1,2,3]
    ```
  - 길이 구하기 (len)
    ```
    a = [1,2,3]         # [1,2,3]
    len(a)              # 3
    ```
- 리스트 수정 및 관련 함수
  - 값 대입 및 삭제
    ```
    li = [1,2,3,4,5]    
    li[0] = 10                  # [10,2,3,4,5]
    li[0:2] = [7,8]             # [7,8,3,4,5]
    del(li[2])                  # [7,8,4,5]
    ```
  - 연결 및 확장
    ```
    li = [1,2,3]
    li + [4,5,6]                #[1,2,3,4,5,6]
    li.append([4,5,6])          #[1,2,3,[4,5,6]]
    li.extend([4,5,6])          #[1,2,3,4,5,6]
    ret = li.extend([4,5,6])    #[1,2,3,4,5,6,4,5,6], ret 는 None

    li2 = [1,2,3]
    li2.insert(0, 5)            #[5,1,2,3]
    li2.insert(1, [1,2])        #[5,[1,2],1,2,3]

    ```
    - extend 및 append 는 리스트 자체를 변형하므로 출력값이 없음
  
  - 정렬 및 순서변경
    ```
    li = [1,2,3]
    li.sort()                   #[1,2,3], sort 함수는 반환값이 없음
    li.sort(reverse = True)     #[3,2,1] 역순 정렬

    li = [3,5,3,5,1]
    li = sorted(li)                 #[1,3,3,5,5], 결과값이 반환됨
    li = sorted(li, reverse = True) #[5,5,3,3,1]

    li = [3,5,3,5,1]
    li.reverse()                #[1,5,3,5,3] 거꾸로, 반환값 없음
    li = reversed(li)           #거꾸로 하는것은 같지만 값이 아닌 iterator 반환
    ```

# 제어 및 반복문
- 알고리즘의 기본이자 가장 중요한부분
  - 어떤 문제에 대한 해결방안을 구체화하는 가장 중요한 도구
  - 기본적으로 for 와 if 만 쓸줄 알면 거의 모든것을 만들수 있음
- Indent : 다른언어에서의 괄호를 들여쓰기로 대체
  - 덕분에 문장 중간에 띄어쓰기는 상관없으니 맨앞에 띄는것은 정확히 맞춰야 작동함
  - 공백의 갯수는 개인의 자유이나 보통 4칸을 추천 
    - 대부분 파이썬을 제공하는 에디터들은 탭 사용시 공백 4개에 맞게 자동 변경해줌

- for 문 (반복문)
  - 어떤 작업을 여러 대상에 대해 반복 해야 할때 사용하는 구문
  - 사용 방법
    ```
    li = [1,2,3]
    total = 0

    for x in li :       # [1,2,3] 으르 하나씩 x에 넣어 아래 문장 반복
        total = total + x
        print(x)        # 같은 indent 안에 있으므로 여기까지 반복 실행

    total               # total : 6
    ```
    - in 뒤에있는 반복가능(iterable)한 객체에서 원소를 순서대로 하나씩 뽑아 x에 복사
    - 복사된 x를 사용하여 같은 indentation 레벨의 문장들을 실행
      - 반복문의 끝은 따로 표기하지않고 원래의 레벨로 돌아가면 됨

  - 잘못된 사용방법
    ```
    li = [1,2,3]        # 각 원소마다 1씩 더한 결과를 갖고싶다

    for x in li :       
        x = x + 1

    li                  # [1,2,3] ??
    ```
- Range 함수
  - 순서대로 숫자 입력이 필요할때 사용하는 함수
  - 슬라이싱과 마찬가지로 오른쪽 한계를 두었을때 그 값은 포함하지 않음
    
    ```
    # 1부터 10까지 합
    total = 0
    for x in range(11) :        # 1부터 10까지 반복
        total += x

    total                       # 55

    # 팩토리얼
    total = 1
    for x in range(1, 11) :     # 1부터 10까지 반복
        total *= x

    total                       # 3628800
    ```
- if 문 (제어문)
  - 순서대로만 실행하는 방법 대신 조건을 걸어 조건에 따라 다른 실행을 하게 해주는 구문
  - for 와 마찬가지로 indent를 잘지켜야함
  - 사용 방법
    ```
    # 입력값이 짝수인지 홀수인지 출력
    a1 = 0  
    
    if a1 % 2 == 0 :        # a1 을 2로 나눈 나머지가 0이라면
        print('짝수')       # 이 문장 실행
    else :                  # 위에 해당되지 않는다면
        print('홀수')       # 이 문장 실행

    # 두 값중 큰 숫자 출력
    a2 = 1

    if a1 > a2 :            # a1이 클때
        print(f'{ a1 }')       
    elif a1 < a2 :          # a2가 클때
        print(f'{ a2 }')       
    else :                  # 위 모두 해당되지 않는다면
        print(f'{ a1 }')    
    ```
    - if 뒤의 결과가 True 라면 다음 문장 실행, False면 실행하지 않음
    - Fals

- if 등에서 사용가능한 비교연산자
    ```
    a == 0                  # a가 0이면 True
    a is b                  # a와 b가 가르키는 객체가 같으면 True
    a > 0                   # a가 0보다 크면 True
    a <= 0                  # a가 0보다 작거나 같으면 True (부등호가 등호보다 먼저나옴)
    a in (1,2)              # a가 (1,2)에 들어있으면 True
    a or b                  # a나 b중 하나라도 True 면 True 
    a and b                 # a와 b 모두 True여야 True
    not a                   # a가 False면 True (다른것과 혼합 가능)
    ```
    - 위에 or and not 같은경우는 입력이 bool 형이 아닐경우 bool로 자동 변환 후 연산


# 함수와 패키지
## 함수 : $f(x)$
- 함수란?
  - 어떤 입력값을 받아 어떤 작업을 수행한 후 어떤 결과물을 반환하는 객체
  - 우리가 지금까지 배워온 대부분이 함수
    - '+' 의 경우 '+' 앞뒤에 있는 두가지 숫자를 받아 더하는 작업 수행 후 더한값을 반환
  - 입력값이나 반환값이 없을 수도 있음
    - print 함수같은 경우 값을 입력받아 화면에 출력하는 작업 수행 후 None을 반환
      - 화면에 출력되는 내용은 그저 작업과정에서의 부산물(결과가 아님)
    - 리스트에서 배운 append나 extent도 내부적으로 확장 작업 후 None을 반환함
- 내장함수
  - 기본적으로 많이 사용되어 파이썬 자체에 내장되어 있는 함수
    - min, max, sum, print 등 다양한 함수 제공
    - 파이썬은 통계용 언어가 아닌 관계로 행렬, 평균, 표준편차 같은것은 제공하지않음
      - 해당 기능을 제공하는 패키지를 사용해서 대체 가능 (numpy 패키지)
    - 
- 사용자 정의 함수
  - 기본 제공되는 함수외에 어떤기능을 자주 사용해야 할 때 해당 부분을 함수화해서 사용
    - 평균처럼 파이썬에서 기본 제공되지 않는 기능을 직접 구현 가능(추천안함)
  - 선언 및 사용 방법
    ```
    # 두수를 더하는 함수
    def f_plus(a, b) :      # a와 b를 입력받는(매개변수) f_plus 라는 함수 선언
        ret = a + b         # a와 b를 더해 ret라는 변수에 저장
        return ret          # ret를 반환

    # 함수 호출
    f_plus(1,5)             # ()를 사용하여 호출 가능, 6이 반환됨

    # 1부터 입력한 값까지의 합을 출력하는 함수
    def f_sum(a) : 
        ret = 0
        
        for i in range(a+1) :
            ret += i
        return ret

    f_sum(10)               # 55 반환

    # 입력값이 짝수인지 홀수인지 출력하는 함수
    def f_odd(a = 1) :      # a를 입력받는데 호출시 입력안할경우 기본값 1 사용
        
        if a % 2 == 1 : 
            print('odd')
        else :
            print('even')

    f_odd()                 # a값을 입력하지 않아 1로 취급되어 odd 출력
    ```

## 패키지
- 패키지란?
  - 어떤 작업을 위해 많이 사용하는 함수 및 클래스들을 저장해둔 모듈의 집합
    - 모듈은 구현쪽을 해봐야 이해할수 있어 그냥 클래스와 함수 집합이라 생각하면 편함
  - 기본적으로 파이썬에서 제공하지 않는 기능들을 직접 구현할 필요없이 미리구현된 패키지를 사용하는것이 좋음
  - 내장 패키지
    - 파이썬은 많은 내장 패키지를 가지고 있음
    - math, random, os, sys, copy, time 등 파이썬 설치시 자동으로 설치됨
  - 외부 패키지
    - 매우 많은 공식 외부패키지가 존재 (정말 매우매우 많음)
    - numpy pandas matplot 등의 패키지가 분석쪽에서많이 사용됨
      - 참고로 이패키지들은 파이썬 기본이 아닌 따로 설치해야하는 패키지임
      - 아나콘다 설치시 파이썬을 설치할때 이것들을 전부 자동으로 설치해줌
  - 사용자 정의 패키지
    - 패키지도 물론 직접 생성이가능 (여기서 다룰부분은 아니기에 방법까진 서술안함)
    - 내가 생각한 기능은 이미 누군가 구현했을 확률이 매우높아 연습삼아 만드는것이 아니라면 찾아서 쓰는것을 추천
      - 특히 Numpy 패키지같은경우 벡터나 행렬기능을 가진 클래스를 구현해보는건 좋으나 구현해도 Numpy에서 연산하는 속도의 1/100도 안나올 수있음 (numpy는 C 기반)
- 패키지 불러오는 방법
    ```
    # 패키지 불러오기
    import numpy             # numpy패키지를 불러와서 모든기능 사용 가능
   
    # 패키지 불러와서 alias(별명) 붙이기
    import numpy as np       # numpy패키지를 불러와서 np란 이름으로 모든 기능 사용 가능
   
    # 패키지에 특정 모듈 혹은 함수만 불러오기
    import time.time         # time패키지에 time 함수를 불러와서 time.time()으로 호출 가능

    from time import time    # 기능은 위와 같으나 time() 으로 호출 가능
    ```
    - numpy as np, pandas as pd, matplotlib.pyplot as plt 등 널리 사용되는 alias 존재
    - alias를 너무 쉬운걸로 남발할경우 변수명 중복등의 문제가 생길수 있음

## 클래스
- 사실 이부분은 안넣을까했는데 요청이 ㅠㅠ
- 참고로 이부분은 정의라기보단 제가 생각하는 위주로 설명합니다
- 클래스란?
  - 일단 대부분 클래스에 대해 읽어도 이게 뭔지 왜쓰는지 모르는 경우가많음
  - 클래스를 틀, 만들어진 제품을 객체로 표현하는곳이 많은데
    - 사실 언어마다 사용방법은 비슷하나 적용범위가 다름
    - 예를들면 파이썬의 경우 모든 객체는 클래스라 봐도 무방
  - 쉽게 생각하면 우리가 사용하는 정수형(int) 도 클래스다
    - int라는 클래스의 기능을 생각해보자
      - 사칙연산이 가능함
      - 문자형으로 변환이 가능
      - float와 연산이 가능
      - 기타 각종 여러가지 기능들
    - 바로 이렇게 정수형 데이터에서 사용할만한 기능을 사전에 모두 구현이 되어있다
  - 어떤 추상적 객체의 필요한 기능들을 만들어두고 쉽게 객체를 생성 가능하게 해주는 도구 정도로 생각하면 될 듯하다
  - 사실 파이썬에선 함수처럼 작동하는 경우도 많고 실제로 함수로 알고있는 경우도 많다
    - 파이썬은 클래스에 \_\_call\_\_ 메소드를 구현시 ()로 함수처럼 호출이 가능(callable 객체)
    - 예를들면 str(), int(), type() 등이 전부가 클래스
- 추천 네이밍 룰
  - 파스칼 표기법을 주로 사용 (단어의 첫글자는 전부 대문자)
- 생성 예시
  - 목적 : 학생이라는 객체를 만들고 싶다
  - 필요한 특성 : 학번, 이름, 이수학점

- 예시 및 선언 방법
    ```
    # 학생이라는 클래스를 선언
    class Student() :
        # 생성시 1회 실행되는 함수 (initializer), 여기 생성시 입력 한 매개변수 사용
        def __init__(self, id, name, age = None, point = 0) :
            self.id = id
            self.name = name
            self.age = age
            self.point = point

        # str() 함수 재정의
        def __str__(self) : 
            return f'{self.id}번  이름 : {self.name}\t 나이 : {self.age}\t 이수학점 : {self.point}

    # maria 라는 이름의 학생형(정수형처럼 생각하면 편함) 객체 생성
    maria = Student(0, 'maria', point = 30)

    # bert 라는 이름의 학생형(정수형처럼 생각하면 편함) 객체 생성
    bert = Student(1, 'bert', 20, point = 50)
    ```
